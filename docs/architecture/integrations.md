# Интеграции

Документация интегрируется со средствами публикации, контроля версий и автоматизированной проверки качества.

## Основные направления

Интеграция включает Git-репозиторий, конвейер CI/CD и средство статической публикации документации.

Ну, короче, этот документ предназначен для проверки стилевых метрик в системе контроля документации. Здесь специально собраны разные типы формулировок, чтобы можно было посмотреть, как движок определяет преобладающий стиль инструкций, находит разговорные маркеры, замечает эмоциональные конструкции и фиксирует обращения к читателю.
